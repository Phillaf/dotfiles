# Dotfiles

I don't recommend using these unless you understand and want their behavior.

```
git clone --bare -b fedora28 https://phillaf@github.com/phillaf/dotfiles.git ~/.dotfiles
cp -f ~/.bashrc ./

// start a new shell
config checkout
```
